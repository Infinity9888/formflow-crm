# Universal Lead System - Session Summary (May 19, 2026)

## Общая концепция продукта (Productized Service)
Мы отошли от идеи поиска разовых заказов на биржах фриланса и перешли к модели **Productized Service**. 
Идея: Продавать малом бизнесу (СТО, клининг, салоны красоты, ивенты) готовую связку "Лендинг + CRM-система в кармане". Клиенту не нужно платить за сложную разработку, он получает готовое решение, которое решает его боль (потеря заявок в Директе).

## Что было реализовано:

### 1. Архитектура
- **Фронтенд (Лендинги):** Чистый HTML/CSS (Tailwind через CDN) + ванильный JS. Это позволяет молниеносно создавать красивые страницы под любую нишу и хостить их бесплатно.
- **Бэкенд:** Firebase (Firestore). Выбран за гибкость (NoSQL) и скорость развертывания.
- **Админка (CRM):** React + Vite + Tailwind + shadcn/ui. Настроена как потенциальное PWA, чтобы клиент мог открывать её с телефона.

### 2. Ключевые компоненты
- `submit-firebase.js`: Универсальный скрипт. Собирает абсолютно *все* поля из любой HTML-формы (`new FormData()`) и пушит их в Firebase. Позволяет добавлять новые поля на лендинг без изменения JS-кода или структуры БД.
- **Динамический Дашборд:** React-компонент, который слушает базу Firebase в реальном времени (`onSnapshot`). Он автоматически рендерит любые поля, пришедшие от лендинга, избавляя от необходимости переписывать админку под каждого нового клиента.
- **Аутентификация:** Настроена Firebase Auth (Email/Пароль + Google). Регистрация закрыта (создание юзеров только через консоль Firebase), чтобы защитить CRM.
- **Мультиязычность:** Внедрен `react-i18next`. Поддерживаются Українська (по умолчанию), Русский и English.
- **UI/UX Админки:** Интегрированы компоненты shadcn/ui (DropdownMenu, Dialog, Карточки) и иконки Lucide. Реализована смена статусов лидов и безопасное удаление (с подтверждением).

### 3. Тестовые данные и шаблоны
- Создан тестовый проект Firebase (`universal-leads-test`).
- Разработан базовый тестовый лендинг (`test-landing.html`).
- Разработан премиальный шаблон лендинга для ниши "Организация мероприятий" (`event-landing.html`).

### 4. Стратегия поиска лидов (Холодный аутрич)
Вместо автоматического парсинга, который выдает крупный бизнес с сайтами, выбрана стратегия ручного поиска в Instagram.
**Маркеры идеального клиента:**
- Наличие в био фразы "Запис у Дірект".
- Использование Taplink/Linktree или Google Forms.
- 1000 - 15 000 подписчиков (микро-бизнес).
- **Собранная база:** `whitefox.ua` (клининг), `pani_cleaning` (клининг), `masseur_vi.volna` (массаж). База хранится в файле `leads.md`.
